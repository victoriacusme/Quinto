Desarrollo web

El desarrollo web es todo lo que implica la creación de un sitio web. Por lo general, se refiere al lado de codificación y programación de la producción del sitio web en lugar del lado del diseño web.

Last updated 3 mins ago

Artificial intelligence

La inteligencia artificial (Artificial Intelligence) es la capacidad que tienen las máquinas para pensar por sí mismas. Se demuestra cuando una máquina puede realizar una tarea, antes realizada por un ser humano y que se considera que requiere la capacidad de aprender, razonar y resolver problemas.

Last updated 20 mins ago

Programación Orientada a Objetos

La programación orientada a objetos es un paradigma que utiliza objetos que contienen datos para organizar modelos en lugar de acciones. Los paradigmas de programación anteriores se centraban en la lógica detrás de las acciones en lugar de los objetos en sí.

Last updated 5 mins ago

SQL

SQL (Structured Query Language en inglés), es un lenguaje de consulta estructurada, de dominio específico, que se usa en programación. Su objetivo es recuperar información de base de datos y hacer cambios cuando sea necesario.

Last updated 3 mins ago

Desarrollo de aplicaciones moviles

Se denomina desarrollo de aplicaciones móviles al conjunto de conocimientos, técnicas y procesos necesarios para crear software simple que pueda ser usado en dispositivos móviles como por ejemplo tabletas y celulares.

Last updated 3 mins ago

Seguridad informática

La Seguridad Informática se refiere a todos los procesos y conocimientos que pueden llevarse a cabo para proteger mediante medidas de seguridad, las bases de datos y toda información privada en los sistemas informáticos.

Last updated 3 mins ago
